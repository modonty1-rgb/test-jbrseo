# JBRSEO Landing Page — Clean Plan

**Goal:** Integrate Modonty Power Points into the JBRSEO landing page so visitors see the exclusive agent offer, the 18-for-12 value, and how Modonty serves them. One clear goal: more consultation leads.

---

## 1. What We’re Selling (from Strategy)

- **Exclusive Agent:** JBRSEO = الوكيل الحصري لمدونتي في السعودية
- **18 for 12:** Pay 12 months, get 18 months delivery
- **Authority Blog:** Content on Modonty.com → faster ranking, stronger backlinks
- **Zero-Touch:** We write, client only approves
- **Transparency:** Client Console + GTM/Analytics
- **DNA Intake:** 100+ questions → content in client’s voice

---

## 2. What We Change

| Where | What |
|-------|------|
| **Hero** | Eyebrow: "الوكيل الحصري لمدونتي في السعودية". One pill: "ادفع 12، احصل على 18". Body: one line on Modonty + 18-for-12. Single CTA: احجز استشارتك. |
| **Services** | One item on "مدونة مركزية (مدونتي)" + لوحة عميل. |
| **Pricing** | One clear line: "18 شهر تسليم مقابل 12 شهر دفع". Keep plans + testimonials. |
| **Features** | One card: لا تكتب، اعتمد فقط. One card: لوحة عميل + أرقامك في Analytics. One block: استمارة 100+ سؤال. |
| **About** | One line: الوكيل الحصري لمدونتي في السعودية. |
| **BEM (كيف نعمل)** | Steps: استلام مقالات شهرية، اعتماد من لوحة واحدة، استمارة استقبال شاملة. |
| **Contact + Final CTA** | Same CTA: احجز استشارتك. Short form. |

---

## 3. Files to Edit

- **`app/content/home.ts`** — All new/updated copy (hero, servicesOverview, pricingSection, featuresGrid, aboutAgency1, bemSection).
- **Components** — Only if we add a new block or visual (e.g. value badge in Pricing, Console screenshot in Features). Otherwise reuse existing Hero, PricingSection, FeaturesGrid, BEMSection, AboutAgency1.

---

## 4. UX Rules (from best practices)

- One primary CTA: "احجز استشارتك" (Hero + Contact + Final CTA).
- 18-for-12 visible in Hero and Pricing.
- Exclusive agent in Hero eyebrow + About.
- No extra primary CTAs; secondary links (e.g. المدونة) stay secondary.
- Mobile: CTA tappable, text readable.

---

## 5. Done When

- [ ] Hero says "الوكيل الحصري لمدونتي" and "ادفع 12، احصل على 18".
- [ ] Services mention مدونتي + لوحة عميل.
- [ ] Pricing shows "18 شهر مقابل 12".
- [ ] Features explain: اعتمد فقط، شفافية لوحة/Analytics، استمارة 100+ سؤال.
- [ ] About + BEM reinforce exclusivity and process.
- [ ] One clear path to contact/consultation.
