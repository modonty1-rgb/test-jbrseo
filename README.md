# JBRSEO Coming Soon

A modern Next.js application featuring a coming soon page with a redirect button to the whitelist registration.

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- ESLint

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js App Router directory
  - `page.tsx` - Coming soon homepage
  - `layout.tsx` - Root layout
  - `components/` - React components
    - `RedirectButton.tsx` - Button component that redirects to whitelist
  - `globals.css` - Global styles with semantic color tokens

## Features

- Responsive design
- Semantic color tokens (CSS variables)
- Accessible button with ARIA labels
- SEO-friendly metadata
- TypeScript strict mode
